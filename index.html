Projects

Here are some examples showcasing my work. Selecting the project titles will bring you to the deployed site!

GoT Theories?

GoT Theories is a fun interactive application for Game of Thrones fans to post their own theories, as well as comment on other users theories. This application was part of a group project in which I was primarily responsible for the front-end design and development. This application uses Materialize, CSS, jQuery, Handlebars.js, Node.js, Express, Firebase, MySQL, and Sequelize.

GifTastic

GifTastic is an enjoyable application that uses the Giphy API to pull gifs of your choosing. The user is greeted with pre-defined buttons for characters from The Office (because c'mon... who doesn't love The Office?). You can choose to view some Michael Scott gifs, or add your own button for gifs you would like to see! The gifs are paused by default and are enabled on click. GifTastic was created using Bootstrap, CSS, and jQuery.

Train Schedule

This is a train schedule application that demonstrates storing data on a Firebase server. When the user adds a new train to the schedule, the new schedule is appended to the list and added to the Firebase Server. This application was created with the help of Materialize, CSS, jQuery, moment.js, and Firebase.
