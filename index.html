Crystal Collector

This is a JavaScript-based web application I created called Crystal Collector. In this game, a random number is generated and random values are applied to each of the four crystals. It is up to the user to add up their total to equal the randomly generated number.

Nomad

Nomad is a one-stop-shop solution for anyone traveling. The user can search any city and Nomad will display the current weather, recent news articles regarding the city, and any events through Ticketmaster. This web application was developed with a team and uses Javascript, Firebase, and APIs.

Train Schedule

This is a train schedule application that demonstrates storing data on a Firebase server. When the user adds a new train to the schedule, the new schedule is appended to the list and added to the Firebase Server.
